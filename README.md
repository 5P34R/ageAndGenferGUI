# ageAndGenferGUI

# Installation

## Step 1
Create a virtual enviroment and activate it

for `cmd`:
```bash
python -m venv venv
.\venv\Scripts\activate
```

for `terminal`:
```bash
python -m venv venv
source venv/bin/activate
```

## Step 2
Install dependencies
```
pip install -r requirements.txt
```

## Step 3 
Lets run it
```
python main.py
```
